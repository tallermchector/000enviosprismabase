# User Guide

Welcome to the Envíos DosRuedas platform. This guide explains the core features available to users and administrators.

## Core Features

### 1. Shipment Quoting

Users can get an instant price estimate for their shipment.

- **How to use**: Navigate to the "Cotizar" section.
- **Service Types**:
  - **Express**: Faster delivery with premium pricing.
  - **Low Cost**: Economical option for non-urgent shipments.
- **Process**: Enter the pick-up and delivery addresses. The system uses real-time routing to calculate the distance and provide an accurate price based on current rates.

### 2. Client Registration

Regular customers can register their details to speed up future orders.

- **Benefits**: Saved addresses and contact information.
- **Requirements**: A valid phone number (used as a primary identifier) and address.

### 3. Order Management

The system allows for the creation and tracking of shipments.

- **Timing Windows**: When placing an order, users specify pick-up and delivery windows. The system ensures that delivery times are scheduled logically after pick-up times.
- **Geocoding**: All addresses are automatically geocoded to ensure accurate routing for delivery personnel.

### 4. AI Testimonial Summary

A tool for administrators to quickly understand customer feedback.

- **Usage**: Provide a list of customer testimonials, and the AI will generate a concise summary highlighting the key sentiments and common themes.

## Frequently Asked Questions

### How is the price calculated?

Pricing is based on the driving distance between the origin and destination. There are fixed price ranges for distances up to 10km, and a per-kilometer rate for longer distances.

### What happens if an address cannot be found?

The system uses OpenStreetMap for geocoding. If an address is not recognized, try adding more specific details like the city (Mar del Plata) or neighborhood.
